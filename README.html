#############DAILY SETUP################

Export AWS credentials to terminal to ensure terraform deploys to correct account and access to secret manager: Credentials will be shared by AWS user with shared account

export AWS_ACCESS_KEY_ID=<accesskey>
export AWS_SECRET_ACCESS_KEY=<secret accesskey>
export AWS_DEFAULT_REGION=eu-west-2

Makefile

  • make requirements (only once or when pip list updated)
  • make dev setup
  • make run-checks

Ensure venv is active and python path exported

  • source venv/bin/activate
  • export PYTHONPATH=$(pwd)

END OF DAY: Terraform destroy!
